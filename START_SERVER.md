# Как запустить проект

## Способ 1: Через локальный сервер (Рекомендуется)

### Python 3:
```bash
cd vcard-personal-portfolio
python3 -m http.server 8000
```
Затем откройте в браузере: `http://localhost:8000`

### Python 2:
```bash
cd vcard-personal-portfolio
python -m SimpleHTTPServer 8000
```

### Node.js (http-server):
```bash
# Установите http-server глобально
npm install -g http-server

# Запустите
cd vcard-personal-portfolio
http-server -p 8000
```

### PHP:
```bash
cd vcard-personal-portfolio
php -S localhost:8000
```

## Способ 2: Прямое открытие файла

⚠️ **Внимание:** При открытии `index.html` напрямую (двойной клик) могут возникнуть проблемы:
- CORS ошибки при загрузке модулей
- Проблемы с относительными путями
- Некоторые функции могут не работать

**Рекомендуется использовать локальный сервер!**

## Проверка работы

1. Откройте консоль браузера (F12)
2. Проверьте, нет ли ошибок
3. Попробуйте кликнуть по навигационным кнопкам
4. Убедитесь, что страницы переключаются

## Решение проблем

### Проблема: "Failed to load resource"
- Убедитесь, что вы используете локальный сервер
- Проверьте пути к файлам (должны быть относительными: `./assets/...`)

### Проблема: Навигация не работает
- Откройте консоль браузера (F12) и проверьте ошибки
- Убедитесь, что `script.js` загружается
- Проверьте, что элементы с `data-nav-link` и `data-page` существуют

### Проблема: Стили не применяются
- Проверьте, что `style.css` загружается
- Убедитесь, что путь к CSS правильный: `./assets/css/style.css`
